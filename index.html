O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação utilizada para criar páginas web. Com ele, você pode organizar seu conteúdo na internet, utilizando uma variedade de elementos como parágrafos, imagens, links, listas e muito mais.

Aprender HTML é o primeiro passo para se tornar um desenvolvedor web. Explore as outras páginas para saber mais sobre meu currículo e exemplos de uso de HTML!
